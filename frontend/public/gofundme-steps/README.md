# GoFundMe Step Screenshots

This directory contains screenshots for the GoFundMe manual wizard step-by-step guide.

## Expected Files

Place screenshots in this directory with the following naming convention:

- `step1-start.png` - GoFundMe homepage with "Start a GoFundMe" button
- `step2-beneficiary.png` - "Who are you fundraising for?" selection page
- `step3-category.png` - Category selection page (Medical, Emergency, etc.)
- `step4-goal.png` - Goal amount input page
- `step5-location.png` - Location (city/state) input page
- `step6-title.png` - Campaign title input page
- `step7-story.png` - Story editor page with formatting tools
- `step8-media.png` - Photo/video upload page
- `step9-review.png` - Review page before publishing
- `step10-publish.png` - Account creation and publish confirmation
- `step11-bank.png` - Bank account connection page
- `step12-share.png` - Sharing options page with social media buttons

## Screenshot Guidelines

- **Resolution**: 1200x900 pixels or higher
- **Format**: PNG or JPG
- **Content**:
  - Show the actual GoFundMe interface clearly
  - Highlight the relevant input field or button
  - Avoid including personal information
  - Use arrows or annotations if helpful

## Fallback Behavior

If screenshots are missing, the GoFundMe wizard component will display:

- Placeholder icon (image placeholder)
- Text instructions only
- The wizard remains fully functional without images

## How to Obtain Screenshots

1. Visit https://www.gofundme.com/c/start
2. Start the fundraiser creation process
3. Take screenshots at each step listed above
4. Crop/resize to focus on the main content area
5. Save with the filenames listed above
